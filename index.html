Tabs

tabs(tabs, active, style, fade, pane_id_prefix)(attributes)

tabs-fade(tabs, active, style, pane_id_prefix)(attributes) 辅助 mixin，启用 fade 效果
tabs-pill-fade(tabs, active, pane_id_prefix)(attributes) 辅助 mixin，使用 pill 样式，启用 fade 效果
  • tabs: 对象或者数组,如果是对象,属性名作为id,属性值作为标签的标题,如果是数组,数组下表作为id,数组元素作为标签的标题
  • active: 字符串或者数字,默认激活的标签面板,如果是 tab 是对象,传入面板对应的属性名,如果是数字,传入面板对应的数组下标
  • style: 字符串, 'tab' 或者 'pill', 默认值为 'pill',标签栏样式,见: navs style
  • fade: 布尔值, 是否启用渐入效果,默认值为 false,见: Fade effect
  • pane_id_prefix: 字符串, 默认值为空字符串,标签面板 id 的前缀
  • attributes: mixin attributes，可选，用来给 mixin 中的 dom 元素添加属性

tab(key)

  • key: 字符串，对应 tabs mixin 中 tabs 参数的属性名或者数组下标
  • block: 传入属性 key 对应的 block

tabs 传入对象

  • tab-title-1
  • tab-title-2

This is tab-1

This is tab-2

+tabs({'tab-id-1': "tab-title-1", 'tab-id-2': "tab-title-2"}, 'tab-id-2', 'pill', true, 'test-object-')
    +tab('tab-id-1')
        p This is tab-1
    +tab('tab-id-2')
        p This is tab-2

tabs 传入数组

  • tab-title-1
  • tab-title-2

This is tab-1

This is tab-2

+tabs-fade(["tab-title-1", "tab-title-2"], 0, 'tab', 'test-array-')
    +tab('0')
        p This is tab-1
    +tab('1')
        p This is tab-2

利用 attributes 给 ul 元素添加属性

  • tab-title-1
  • tab-title-2

This is tab-1

This is tab-2

+tabs-fade(["tab-title-1", "tab-title-2"], 0, 'tab', 'test-array-')(ul={'class':'col-sm-offset-2'})
    +tab('0')
        p This is tab-1
    +tab('1')
        p This is tab-2
Panels

panel(type, title)(attributes)

mixin panel-default(title) 辅助 mixin，设置样式为 panel-default
mixin panel-primary(title) 辅助 mixin，设置样式 panel-primary
mixin panel-success(title) 辅助 mixin，设置样式 panel-success
mixin panel-info(title) 辅助 mixin，设置样式 panel-info
mixin panel-warning(title) 辅助 mixin，设置样式 panel-warning
mixin panel-danger(title) 辅助 mixin，设置样式 panel-danger
  • type: 字符串，指定标题颜色，取值参见 Contextual alternatives
  • title: 字符串，文字作为标题，无样式
  • attributes: mixin attributes，可选，用来给 mixin 中的 dom 元素添加属性

panel-heading()

  • block: 传入 panel 标题的 block

panel-body()

  • block: 传入 panel 内容对应的 block

panel-body-append()

  • block: 传入 panel 内容后面附加的表格或者列表的 block，见 With tables、With list groups

panel-footer()

  • block: 传入 panel 脚注对应的 block

简单示例

panel 标题
panel 正文
+panel-success('panel 标题')
    +panel-body panel 正文

    +panel-footer
        a.panel-title(href='#') 链接作为 panel 脚注
Tables

table(header, data, type, auto_index)(attributes)

mixin table-indexed(header, data, type) 辅助 mixin，自动在第一行添加行号
mixin table-striped(header, data, auto_index) 辅助 mixin，设置样式 table-striped
mixin table-bordered(header, data, auto_index) 辅助 mixin，设置样式 table-bordered
mixin table-hover(header, data, auto_index) 辅助 mixin，设置样式 table-hover
mixin table-condensed(header, data, auto_index) 辅助 mixin，设置样式 table-condensed
  • header: 对象数组，格式见示例
  • data: 对象数组，格式见示例
  • type: 字符串或者数组，表格样式，刻意省略'table-'，见Tables
  • auto_index: 布尔值，自动在每行生成行号
  • attributes: mixin attributes，可选，用来给 mixin 中的 dom 元素添加属性

简单示例

2Jacob2Thornton2@fat
3Larry3the Bird3@twitter

Modals

mixin modal(id, title, content, close_text)(attributes)

  • id: 字符串，modal 的 id
  • title: 字符串，标题
  • content: 字符串，正文
  • close_text: 字符串，关闭按钮文字
  • attributes: mixin attributes，可选，用来给 mixin 中的 dom 元素添加属性

modal-header()

  • block: 传入 modal 标题的 block

modal-body()

  • block: 传入 modal 正文的 block

modal-footer()

  • block: 传入 modal 脚注 block
